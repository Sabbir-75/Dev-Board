Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM :

  • getElementById();
  • getElementsByClassName();
  • querySelectorAll();
  • querySelector();

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML returns tag and visible text content.
2. innerText returns just visible text content but doesn't consider hiddden element.
3. textContent property is used to set or return the text value of the selected node and all its descendants.

Question-3: What is event delegation in the DOM?

Event Delegation refers to the process of handaling events at a heigher level in the DOM than the element on which the event originated using event propagation.

Question-4: What is event bubbling in the DOM?

At first where an event triggered on a child element propegates upward through its ansestor in the Document Object model.
